# 個人用勤務管理フォーム

## 概要

個人・ローカルで使用する想定の勤務管理フォームです。

- Backend: Python, Flask

- Frontend: TypeScript, React, React Hook Form


## 実行

```
docker compose build
docker compose up  
```
